Put your game files in this folder:

-Wolfenstein 3D shareware -> *.WL1 (already included)
-Wolfenstein 3D -> *.WL6
-Spear of Destiny demo -> *.SDM
-Spear of Destiny -> *.SOD
-Mission 2: Return to Danger -> *.SD2 (see below)
-Mission 3: Ultimate Challenge -> *.SD3 (see below)
-Super 3D Noah's Ark -> *.N3D

----------
Starting the game normally will start the last one you played.
Starting the game by holding START will open the "game selector" screen.
If the selected game isnt found, you'll be sent back to the SNES UI.

----------
Spear of Destiny, "Mission 2 and Mission 3 share some identical files:
AUDIOHED.SOD
AUDIOT.SOD
CONFIG.SOD
VGADICT.SOD
VGAGRAPH.SOD
VGAHEAD.SOD

And 3 unique files:
GAMEMAPS.SOD
MAPHEAD.SOD
VSWAP.SOD

You need to rename these 3 files into *.SD2 for Mission 2 and into *.SD3 for Mission 3. So if you have the 3 games, you should have:
AUDIOHED.SOD
AUDIOT.SOD
CONFIG.SOD
GAMEMAPS.SD2
GAMEMAPS.SD3
GAMEMAPS.SOD
MAPHEAD.SD2
MAPHEAD.SD3
MAPHEAD.SOD
VGADICT.SOD
VGAGRAPH.SOD
VGAHEAD.SOD
VSWAP.SD2
VSWAP.SD3
VSWAP.SOD
